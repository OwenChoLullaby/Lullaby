btn-toggle 클릭시 btn-toggle에 class active 추가, toggle-target에 class active 추가
btn-toggle.active 클릭시 btn-toggle에 class active 삭제, toggle-target에 class active 삭제

접근성

패널이 확장된 경우 : btn-toggle의 aria-expanded="true", 닫힌 경우 aria-expanded="false"
btn-toggle 의 id 값과 toggle-target 의 aria-labelledby 값이 같아야함
btn-toggle 의 aria-controls의 값과 toggle-target의 id 값이 같아야함
